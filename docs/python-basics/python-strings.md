# Python 字符串与转义

## 转义字符

字符串中可以使用反斜杠 `\` 表示转义：反斜杠后面的字符不再按原本字面意义解释。

常见例子：

| 写法 | 含义 |
| --- | --- |
| `\n` | 换行 |
| `\t` | 制表符 |
| `\'` | 单引号 |
| `\"` | 双引号 |
| `\\` | 反斜杠本身 |

## 八进制、十六进制和 Unicode

反斜杠后也可以跟字符编码：

```python
print("\141")      # 八进制，输出 a
print("\x61")      # 十六进制，输出 a
print("\u4f60")    # Unicode，输出 你
```

## 原始字符串

如果不希望字符串中的反斜杠被当成转义符，可以在字符串前加 `r`，创建原始字符串（raw string）。

```python
s1 = r"\'hello, world!\'"
s2 = r"\n\\hello, world!\\\n"

print(s1)
print(s2)
```

输出：

```text
\'hello, world!\'
\n\\hello, world!\\\n
```

## 记忆要点

- 普通字符串中，`\` 会开启转义。
- 想输出反斜杠本身，需要写成 `\\`。
- 正则表达式、Windows 路径等场景经常使用原始字符串。
